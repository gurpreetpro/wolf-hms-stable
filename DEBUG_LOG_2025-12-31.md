## 1. Core System Stabilization (Atomic Simulation)
**Goal:** Fix failing atoms in the core clinical loop (OPD -> Ward -> Care -> Discharge).

### Critical Fixes
*   **Nursing Module (500 Errors):**
    *   **Root Cause:** Schema mismatch. Nursing tables (`pain_scores`, `fluid_balance`) expected `patient_id` (UUID) but controllers were inconsistent.
    *   **Fix:** Implemented "Hybrid Schema" logic (Admission ID = Integer, Patient ID = UUID) across all Nurse Controllers.
    *   **Fix:** Standardized `wound_assessments` and `fall_risk` endpoints to use the central `ResponseHandler` and correct column names (`recorded_by` vs `assessed_by`).
*   **Discharge Module (400 Errors):**
    *   **Root Cause:** "Pending Department Clearance". Patients could not be discharged due to open Tasks or Lab Requests.
    *   **Fix:** Automated "Care Task Completion" in the simulation script to close all open tasks before attempting discharge.

### Result
